Help: Forum

All forum posts are sanitized, meaning all HTML is escaped. That being said, there are a few features you can use:

URLs

Any URL starting with http:// will automatically be converted into a link.

BBCode / DText


Post Link

BBCode: [post]12345[/post]
Description: Link to a specific post number, where 12345 would be the id number of the post.
Result: Post #12345

Forum Link

BBCode: [forum]12345[/forum]
Description: Link to a forum number, where 12345 would be the id number of the topic.
Result: Forum #12345

Bold

BBCode: [b]Bold Text[/b]
Description: Bold a part of your message.
Result: Bold Text

Italic

BBCode: [i]Italicized Text[/i]
Description: Italicize a part of your message.
Result: Italicized Text

Strikethrough

BBCode: [s]Struck-out Text[/s]
Description: Stike-through a part of your message.
Result: Struck-out Text

Spolier

BBCode: [spoiler]The spoiler is a lie.[/spoiler]
Description: Hide text that can be shown by clicking it.
Result: The spoiler is a lie.

Quote

BBCode: [quote]Someone said: Some stuff about cats[/quote]
Description: Quote part of all of what someone said.
Result:

Someone said: Some stuff about cats
