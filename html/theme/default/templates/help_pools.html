Help: Pools

A pool is a group of posts for any purpose (organization, series, etc.) Public visibility allows all users to add posts to your pool, while private allows just you to manage it.

Creating a pool

While in pools, you can create a new pool by clicking the "Create" link in the 3rd level navbar.

Adding a post to a pool

While viewing any post, you can simply click on the "Add to Pool" link on the left sidebar and then select your desired pool.

Editing pool options

While viewing a pool, you can change the options such as name, visibility, etc by clicking the "Edit" link in the 3rd level navbar.

Note: You must have access to edit the pool for this to work.

Editing pool order / Delete

While viewing a pool, you can change the order by clicking the "Editor" link in the 3rd level navbar.

To change the order of a pool using drag and drop, select the "Reorder" mode in the dropdown for mode.

To change the order of a pool by editing order number, select the "Manual Reorder" mode in the dropdown for mode.

To delete a post from a pool, select the "Delete" mode in the dropdown for mode.

Note: You must have access to edit the pool for this to work.

Viewing a list of pools

While in pools, you can view a list of all pools at any time by clicking "All Pools" in the 3rd level navbar.
